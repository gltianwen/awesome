# Python Web Project

This project implement WebServices Call (RESTful), Operate Database(MariaDB) CRUD, config log and databse.


## Project requirements

+ Python3.7
+ MariaDB 10.4
+ aiohttp 3.6.2
+ aiomysql 0.0.20
+ Jinja2 2.11


## Running the Project

----
cd app.py folder first
python3 app.py &
